똑똑한 여우의 5가지 필살기

비즈니스 이메일

상대방의 기분은 상하지 않게, 하지만 할 말은 다 하는 마법의 필터링. '분노조절 이메일'이 당신의 사회생활을 지켜드립니다.

엑셀 수식 / 매크로

"이거랑 이거 합쳐서 보여줘"라고 말만 하세요. 복잡한 VLOOKUP부터 자동화 매크로까지 즉시 복사 가능한 코드로 짜드립니다.

개발 / SQL / 정규식

구글링에 1시간 쓰지 마세요. 시니어 개발자가 옆에서 알려주듯 SQL 쿼리와 머리 아픈 정규표현식을 한 번에 해결해 드립니다.

마케팅 / SNS 카피

조회수를 부르는 인스타그램 해시태그와 클릭을 부르는 광고 카피라이팅. 당신의 제품에 강력한 매력을 불어넣으세요.

인간관계 (거절/사과)

어려운 부탁을 거절해야 할 때, 진심 어린 사과가 필요할 때. 감정 소모는 줄이고 관계는 지키는 최적의 문장을 찾아드립니다.
